1. Information We Collect

1.1 Information You Provide

Data Type Purpose Required
Email Address Account creation, notifications For registered users
Uploaded Media To display on your AR cards Yes, for card creation
Card Metadata File names, sizes, types Automatic with uploads
Payment Information Processing subscriptions For paid plans only

1.2 Automatically Collected Information

  • IP Address: For security and analytics
  • Device Information: Browser type, operating system
  • Usage Data: Card scans, feature usage, error logs
  • Location Data: Approximate location from IP (country/city level)

1.3 QR Code Scan Data

When someone scans your QR code, we collect:

  • Timestamp of scan
  • Scanner's IP address (for analytics)
  • Browser/device type
  • No personal identification of scanner

2. How We Use Your Information

  • To provide and maintain the Service
  • To process payments and prevent fraud
  • To send service-related notifications
  • To improve and optimize our platform
  • To comply with legal obligations
  • To provide card analytics to creators

3. Data Storage & Security

3.1 Storage Locations

  • Media files: Supabase storage (may be in multiple regions)
  • Database: Supabase PostgreSQL
  • Server: Railway.app hosting

3.2 Security Measures

  • Encrypted data transmission (HTTPS)
  • Secure password hashing
  • Regular security updates
  • Access controls and monitoring

3.3 Data Retention

  • Account data: Until deletion request
  • Uploaded media: Until card deletion
  • Analytics data: 24 months maximum
  • Backup data: 30 days maximum
